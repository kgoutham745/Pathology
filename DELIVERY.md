## 🏗️ Architecture

```
┌─────────────────────────┐
│   Web Browser           │
│   (React + Vite)        │
│   http://localhost:3000 │
└────────────┬────────────┘
             │ API Calls (Axios)
             ↓
┌─────────────────────────┐
│   Express.js Server     │
│   http://localhost:5000 │
│   (Node.js Backend)     │
└────────────┬────────────┘
             │ Queries
             ↓
┌─────────────────────────┐
│   MongoDB Database      │
│   Reports, Patients,    │
│   Tests, Lab Settings   │
└─────────────────────────┘
```

6. **Blood Sugar** - Glucose Tests (2 params)
7. **Urine** - Urine Routine (5 params)
